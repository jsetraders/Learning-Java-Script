The JavaScript Syntax

The JavaScript Syntax is how the Java instructions are put together.

#1 JavaScript Values

The JavaScript syntax defines two types of values:
• Fixed values, which are called Literals.
• Variable values, which are called Variables.

#2 JavaScript Literals

The two most important syntax rules for fixed values are:
1. Numbers are written with or without decimals:
• 10.50
• 1001
2. Strings are text, written within double or single quotes.
• "John Doe"
• 'John Doe'

#3 JavaScript Variables

Variables are used to store data values. In JavaScript var, let, const are use to declare variables.
