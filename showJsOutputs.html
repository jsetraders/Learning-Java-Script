Java Script outputs

#1 Using innerHTML

The p HTML element is identified as the interested party to the js script and the content of its ID attribute is output to the screen


#2 Using document.write()

With "document.write()" there is a direct output to the screen without the need for a container.


Using "document.write(5+6)"" and clicking the button will erase all HTML from the page. It will only display the result of the caulculation in the parenthese.

The document.write() method should only be used for testing.


#3 Using window.alert()

With "window.alert(20+9)" there is again a direct output to the screen without the need for a container.

#4 Using console.log()

For debugging purposes, you can call the "console.log()" method in the browser to display data.

#5 JavaScript print

JavaScript does not have any print object or print methods. You cannot access output devices from JavaScript.

The only exception is that you can call the "window.print()" method in the browser to print the content of the current window to the computers printer.
